# Link

节点之间的链接线

## 属性

| 选项         | 描述          | 类型       | 默认值 |
|------------|-------------|----------|-----|
| id         | id          | `string` |     |
| origin     | 输出插槽        | `object` |     |
| target     | 输入插槽        | `object` |     |
| valueType  | 数据传递代表类型字符串 | `string` |     |
| index      | 节点位置        | `number` | 0   |
| originNode | 起点node      | `object` |     |
| targetNode | 终点node      | `object` |     |
